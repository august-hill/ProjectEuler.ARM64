# Project Euler Solutions - ARM64 Assembly (AArch64)

100 [Project Euler](https://projecteuler.net/) solutions in ARM64 assembly, part of a cross-language benchmark suite with standardized benchmarking across 11 languages.

See [ProjectEuler.Benchmarks](https://github.com/august-hill/ProjectEuler.Benchmarks) for cross-language performance comparisons.

## Prerequisites

- **Assembler**: Apple `as` (Clang integrated assembler)
- **Linker**: `cc` (Apple Clang)
- **Platform**: macOS Apple Silicon only

## Running

```bash
# Run a single solution
cd problem_001
as -o solve.o solve.s
cc -O2 -o main_bench main.c solve.o -lm
./main_bench

# Run all benchmarks (from repo root)
./run_all.sh
```

## Project Structure

```
ProjectEuler.ARM64/
  bench.h                   # Shared benchmarking header
  problem_001/
    solve.s                 # Assembly solution
    main.c                  # C bench wrapper calling _solve
  problem_002/
    solve.s
    main.c
  ...
  problem_100/
```

Each `solve.s` exports a `_solve` function returning the answer in register `x0`. The `main.c` wrapper handles timing via `bench.h` and outputs results in a standardized format consumed by the Benchmarks aggregator.

Note: Some complex problems are implemented in C only (no `solve.s`) where hand-written assembly would be impractical.
